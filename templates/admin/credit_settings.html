Configuración de Créditos

Sistema de costos para revendedores

¿Cómo funciona?

Modo "Por día"

Se descuenta X créditos por cada día de duración del usuario.

Ej: 30 días × {{ config.cost_per_day }} créditos = {{ 30 * config.cost_per_day }} créditos

Modo "Por usuario"

Se descuenta un costo fijo por cada usuario creado, sin importar los días.

Ej: 1 usuario = {{ config.cost_per_user }} créditos
